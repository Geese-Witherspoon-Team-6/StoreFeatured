## Development
From within the root directory:

### 1) Install Dependencies
```sh
npm install
```

### 2) Seed Database:
```sh
npm run seed
```

### 3) Run Webpack:
```sh
npx webpack
```

### 4) Start Server:
```sh
npm start
```

### Run Tests

```sh
npm test
```
